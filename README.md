# Funngro
Funngro is a modern and responsive smart website designed for teenagers to explore real-world projects and opportunities. The project is built using HTML, CSS, and JavaScript with a mobile-friendly layout and SEO optimization.

# Features
- Mobile Responsive
- Glassmorphism UI
- Teen Project Page
- Search Functionality
- SEO Optimized

# Technologies Used
- HTML5
- CSS3
- JavaScript
